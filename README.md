# Timtu Store

Una descripción de nuestra e-commerce

## Empezando

Primero clona el repositorio:

```bash
git clone https://github.com/timtu-designs/timtu-store-backend
```

Luego instala las dependencias:

```bash
yarn install
# o
npm install
```

Por último ejecuta el servidor de desarrollo:

```bash
yarn develop
# o
npm run develop
```
